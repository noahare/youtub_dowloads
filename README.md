-**hight controle**:view all avelable format of sound and video and you choose what you like and more you choice path of directory.

-**Flexibility**: file setup.py and all other file work in **linux**,**windows**,**android**.

-**hight Error controle**: many buges has except the code can continue In any case.

-**log**: log file write all bug info and programme Workflow.

-**setup.py**: this file dowloads all tools need to start yt.py.

-**yt.py**: this file the main file .

-**extra_info.py**: read json file and get import info.
